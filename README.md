# haoyi.li

Personal technical webpage for Haoyi Li, an AI Agent Engineer at Alibaba Group.

## Setup

- Install dependencies: `npm install`
- Start development server: `npm run start`
- Build production site: `npm run build`

## Deployment

This repository includes a GitHub Actions workflow to build the site on pushes to `main` and publish the output to the `gh-pages` branch.

Configure GitHub Pages in your repository settings to serve from the `gh-pages` branch.
